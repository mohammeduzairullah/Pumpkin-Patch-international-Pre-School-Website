# Pumpkin Patch International Pre-School Website

An energetic, modern, and highly interactive web platform built specifically for **Pumpkin Patch International Pre-School** in Nalgonda. Designed following the **International Early Years Curriculum (IEYC)** framework guidelines, this website serves as a premium digital gateway for parents exploring early childhood education tracks.

### 📐 The Waterfall Development Approach
The engineering pipeline followed a definitive **Waterfall Model**, driving the project through rigid, sequential phases to guarantee structure and client alignment:
1.  **Requirements Analysis:** Mapping out the school's structural needs (Nursery, LKG, UKG, Daycare, and Summer Camp catalogs).
2.  **Design Phase:** Wireframing custom color blocks and typography configurations inside Figma.
3.  **Implementation & Coding:** Translating fixed, approved canvas elements directly into clean web architecture.
4.  **Final Deployment:** Launching a fully accessible production site.

> 📝 **Note on Maintenance:** Because the core design structure has been formally approved, any post-deployment updates or minor adjustments are treated as standard operational service updates rather than iterative design sprints.
